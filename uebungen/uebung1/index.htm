I. Elementare Bilderzeugung in Java mit ImageJ

0) Installieren Sie das Programm ImageJ.

1) Starten Sie das Programm ImageJ, öffnen Sie einige Bilder (z.B. Bear.jpg) und machen Sie sich mit der Bedienung des Programms vertraut.
Eine Bedienungsanleitung zu ImageJ findet sich hier.

2) Als nächstes soll ein PlugIn von ImageJ installiert werden.
Kopieren Sie hierzu das Beispiel-PlugIn zur Generierung von Farbbildern (GLDM_U1.java ) in das PlugIns/GLDM-Verzeichnis von ImageJ.
Nehmen Sie es in Betrieb, in dem Sie es mit "Compile and Run" (aus dem Plugins-Menu) übersetzen und ausführen.

3) Nehmen Sie dieses PlugIn als Ausgangsbasis für Ihre eigene Arbeit und arbeiten Sie ab jetzt mit Eclipse und starten Sie das Programm auch aus Eclipse heraus.
Benennen Sie diese Datei mit dem Refactoring-Tool von Eclipse in "GLDM_U1_S012345.java" (entsprechen ihrer Matrikelnummer) um.
Bevor Sie die Aufgaben a) bis h) in Java umsetzen, machen Sie sich jeweils Zeichnungen, anhand derer Sie Ihre Programmierung planen können.

Realisieren Sie jetzt ein PlugIn, mit dem Sie die folgenden Aufgaben realisieren können:

Erzeugung

a) eines schwarzen Bildes (schon fertig)
b) eines gelben Bildes
c) eine französische Fahne,
d) eines horizontalen Schwarz/Weiß-Verlaufs,
e) eines horizontalen Schwarz/Rot Verlaufs bei gleichzeitigem vertikalen Schwarz/Blau Verlauf,
f) eine bahamische Fahne,
g) eine japanische Fahne
h) Bonus: eine japanische Fahne mit weichen Übergängen

Bitte kommentieren Sie Ihren Code!

Eine (ältere) Anleitung für das Arbeiten mit ImageJ und Eclipse finden Sie hier.
