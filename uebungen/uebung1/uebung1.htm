I. Elementare Bilderzeugung in Java mit ImageJ

1) Nehmen Sie das die Datei (GLDM_U1.java ) als Ausgangsbasis für Ihre eigene Arbeit und arbeiten Sie ab jetzt mit Eclipse und starten Sie das Programm auch aus Eclipse heraus.
Benennen Sie diese Datei mit dem Refactoring-Tool von Eclipse in "GLDM_U1_S012345.java" (entsprechen ihrer Matrikelnummer) um.
Bevor Sie die Aufgaben a) bis g) in Java umsetzen, machen Sie sich jeweils Zeichnungen, anhand derer Sie Ihre Programmierung planen können.

2) Binden Sie die Datei ij.jar als Projektabhängigkeit ein

Realisieren Sie jetzt ein PlugIn, mit dem Sie die folgenden Aufgaben realisieren können:

Erzeugung

a) eines schwarzen Bildes (schon fertig)
b) eines gelben Bildes
c) einer belgischen Fahne
d) einer Fahne der USA (ohne Sterne)
e) eines horizontalen Schwarz/Rot Verlaufs bei gleichzeitigem vertikalen Schwarz/Blau Verlauf
f) einer tschechichen Fahne
g) einer bangladeschischen Fahne

Bitte kommentieren Sie Ihren Code!
