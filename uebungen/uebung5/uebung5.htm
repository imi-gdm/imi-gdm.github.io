                 Übung  5

einfache Filter / Faltungen

1) Kopieren Sie das Beispiel-PlugIn (GRDM_U5.java) in Ihr Pluginverzeichnis von ImageJ.
Alternativ können Sie diese Übung auch in HTML realisieren. Weitere Hinweise zur Bildverabeitung mit HTML5 finden Sie als Paper und hier das einfache Beispiel des Negativs.

2) Nehmen Sie dieses PlugIn als Ausgangsbasis für Ihre eigene Arbeit. Ändern Sie den Namen der Datei mit Refactor in einen Namen "Uebung5_S012345.java" (entsprechend ihrer Matrikelnummer).

Realisieren Sie jetzt ein PlugIn, mit dem sie die folgenden Aufgaben realisieren können:

a) Original des Bildes (schon fertig)

b) Weichgezeichnetes Bild 3x3 Mittelwertfilter

c) Hochpassgefiltertes Bild

d) Bild mit verstärkten Kanten

Bearbeiten Sie zunächst alle Bildpunkte ohne die Randpixel. Wenn dies funktioniert, fügen Sie eine Randbehandlung hinzu, so dass alle Pixel gefiltert werden können.

 

Abgabe der Aufgaben in der nächsten Übung. Bitte kommentieren Sie Ihren Code.
