Bildvergrößerung in Java mit ImageJ

1) Starten Sie dann das Programm ImageJ, öffnen Sie das Bild component.jpg und schauen Sie, wie das Bild nach einer Vergrößerung mit unterschiedlichen Interpolationsverfahren aussieht.

2) Als nächstes soll wieder ein PlugIn von ImageJ realisisiert werden.
Kopieren Sie hierzu das Beispiel-PlugIn (Scale_.java ) in das PlugIn/Demos-Verzeichnis von ImageJ.
Nehmen das PlugIn in Betrieb, in dem Sie es mit "Compile and Run" (aus dem Plugins-Menu) übersetzen und ausführen. Realisieren Sie jetzt eine main-Methode, so dass Sie das Plugin auch von Eclipse starten können. Orientieren Sie sich hierbei an den letzten Übungen.

3) Nehmen Sie dieses PlugIn als Ausgangsbasis für Ihre eigene Arbeit. Legen Sie eine Kopie der Datei "Scale_.java" mit dem Namen "Scale_S012345_.java" (entsprechend Ihrer Matrikelnr.) an. Realisieren Sie ein PlugIn, mit dem Sie folgende Aufgaben realisieren können:

- Erzeugung eines neuen Bildes mit neuer vorgebarer Bildgröße.
- Hierbei soll die Art, wie die Pixel des neuen Bildes berechnet werden sollen, nach drei Methoden auswählbar sein.

Achtung: Gehen Sie grundsätzlich so vor, dass Sie ausgehend von allen Pixelpositionen des neuen Bildes neue Werte aus dem Ausgangssbild berechnen.

a) Kopie (schon fertig): Hierbei sollen die Pixel des Ausgangsbildes einfach in das neue Bild kopiert werden. Hierbei ist zu beachten, das der Rand des neuen Bildes nicht überschrieben wird, falls das neue Bild kleiner als das Ausgangsbild ist. Ist das neue Bild größer, so sollen die größeren Bereiche einfach freigelassen (schwarz) werden.

b) Pixelwiederholung: Das Ausgangsbild soll auf die neue Größe gebracht werden. Als Interpolationsmethode soll die einfache Pixelwiederholung verwendet werden.

c) Bilineare Interploation: Wie 2, allerdings ist hierbei die bilinare Interpolationstechnik zu verwenden. (Sollte schöner aussehen, evtl. Kontrolle über Photoshop oder ImageJ: Skalierung mit Interpolation)

d) Wer noch Zeit hat, kann versuchen zusätzlich auch noch eine Drehung des Bildes um einen vorgebbaren Winkel zu realisieren. (Dieser Teil ist freiwillig.)
